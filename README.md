# SPMS – Student Project Management System

![SPMS Banner](https://via.placeholder.com/1200x300?text=Student+Project+Management+System)

## Overview

SPMS (Student Project Management System) is a comprehensive web application built to manage academic projects, tasks, and student progress with robust role-based access control. It serves as a unified platform for students, faculty, and administrators to streamline project tracking, task assignments, and overall academic collaboration.

## Features

- **Role-Based Access Control**: Secure interfaces tailored for Students, Faculty, and Administrators.
- **Project Tracking**: Easily manage and monitor the progress of various academic projects.
- **Task Management**: Assign, update, and track tasks for individuals and teams.
- **Interactive Dashboards**: Data visualizations built with Recharts.
- **Modern UI/UX**: Responsive and sleek design powered by Tailwind CSS, featuring dark mode by default.

## Tech Stack

- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **Routing**: React Router v6
- **Styling**: Tailwind CSS & Autoprefixer
- **Form Management**: React Hook Form
- **Notifications**: React Hot Toast
- **Icons**: Lucide React
- **Data Visualization**: Recharts
- **Date Utility**: date-fns

## Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Yashvi-167/SPMS.git
   cd SPMS
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open in browser:**
   Navigate to `http://localhost:5173` to see the application running.

## Building for Production

To create a production-ready build, run:
```bash
npm run build
```
This will compile and optimize the assets into the `dist` directory. You can preview the production build using:
```bash
npm run preview
```
