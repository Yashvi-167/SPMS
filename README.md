## ✨ Features

- 🔐 **Role-Based Access Control**: Tailored dashboards and permissions for Students, Faculty, and Administrators.
- 📊 **Interactive Dashboards**: Real-time data visualizations and insights powered by Recharts.
- 📝 **Project & Task Tracking**: Easily assign, monitor, and update the progress of academic projects and granular tasks.
- 🎨 **Modern UI/UX**: A sleek, responsive design featuring a beautiful dark mode built with Tailwind CSS.

## 🛠️ Tech Stack

### Frontend Core
- **Framework**: [React 18](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Routing**: [React Router v6](https://reactrouter.com/)

### Styling & UI
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Visuals**: [Recharts](https://recharts.org/)

### State & Forms
- **Forms**: [React Hook Form](https://react-hook-form.com/)
- **Alerts**: [React Hot Toast](https://react-hot-toast.com/)
- **Utilities**: [date-fns](https://date-fns.org/)

## 🚀 Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

Make sure you have Node.js installed (Version 16+ recommended).

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Yashvi-167/SPMS.git
   cd SPMS
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Experience the app:**
   Navigate to `http://localhost:5173` in your browser.

## 📦 Building for Production

Ready to deploy? Create an optimized production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```
